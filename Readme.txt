 Command to execute code:
 Change directory to 19_2012A7PS083P_2012A7PS061P/
 
 Then run the following command in a terminal:

 python lexer.py [filename]
 or
 python3 lexer.py [filename]
 in case python3 is not the default python installation
 
 where the code file [filename] can be: p1.txt, p2.txt, p3.txt, p4.txt, p5.txt, p6.txt
 
 If the output scrolls down too fast, and you are unable to scroll up, a timed wait is introduced in another file - time_lexer.py (Command format remains the same)
 
Preferably use a Linux-based OS, since the assignment was completely done on one.  It will work on Windows/other OS too, with a proper Python3 installation.
